Euler's Totient and the Square Root Trick

Many number theory problems can have their complexity reduced down to computing values and sums from the Euler’s totient function. With the right mathematical tricks, these computations can be reduced to near-linear or even sublinear time.

What is the totient?

Euler’s totient function, or phi ϕ(n)\phi(n)ϕ(n) for short, is the count of values less than nnn that are also coprime to nnn. For example, 999 shares factors with 333 and 666. The other six numbers less than 999 are all coprime to 999, so ϕ(9)=6\phi(9) = 6ϕ(9)=6.

The straightforward implementation of phi---checking each value less than nnn for common factors---quickly becomes unwieldy. Only checking values up to n\sqrt{n}n​ is valid, but if a large number of phi values are needed, the complexity is still above linear.O(n3/2)\mathcal{O}(n^{3/2})O(n3/2) when computing all values from 111 to nnn.
